# Security Policy

## Supported versions

Actualmente se da soporte de seguridad a la rama principal (`main`) y a la
última versión publicada.

## Reportar vulnerabilidades

No abras issues públicas para reportes de seguridad.

Envía el reporte con:

- descripción del problema
- pasos de reproducción
- impacto estimado
- versión afectada

a: `security@imagefusion.local`

## Proceso de respuesta

1. Confirmación inicial en máximo 72 horas.
2. Análisis técnico y clasificación de severidad.
3. Definición de fix y ventana de publicación.
4. Publicación de parche y nota de seguridad.
