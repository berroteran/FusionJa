## [1.0.0] - 2026-05-11

### Added

- Base JavaFX app for image composition.
- Layer management with max 4 images.
- Drag-and-drop positioning and canvas auto-growth.
- Merge layers into single image.
- Export PNG/JPG with configurable compression.
- DPI normalization and metadata writing.
- Splash screen and standard menu layout.
- Initial unit tests for composition service.
